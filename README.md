# Kristof Portfolio Dashboard

Dagelijks momentum dashboard voor persoonlijke portefeuille.

## Deploy in 5 stappen

1. Maak een nieuwe GitHub repo aan: `portfolio-dashboard`
2. Upload alle bestanden uit deze map naar die repo
3. Ga naar vercel.com → "Add New Project"
4. Verbind je GitHub repo `kristofgeskens-lab/portfolio-dashboard`
5. Klik "Deploy" — Vercel detecteert Next.js automatisch

## iPhone installatie

1. Surf op Safari naar je Vercel URL (bv. `portfolio-dashboard.vercel.app`)
2. Tik op het Deel-icoontje (vakje met pijltje omhoog)
3. Kies "Zet op beginscherm"
4. Geef het de naam "Portfolio" → Voeg toe

## Data

Koersen worden opgehaald via Yahoo Finance (gratis, geen API key).
Cache: 1 uur. Klik "⟳" om handmatig te verversen.
